nxt-navigation
==================

> Lab 3 of ECSE 211 at McGill, by AJ Ostrow and Vitaliy Kuzmin, Fall 2013. 

### shell commands

To clean the project:

```
$ source clean.sh
```

To clean and compile:

```
$ source compile.sh
```

To clean, compile and uplaod to NXT brick:

```
$ source upload.sh
```
